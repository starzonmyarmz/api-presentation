Fetch

  • A global method that provides an easy, logical way to fetch resources asynchronously across the network.
  • Uses Promises to handle results/callbacks.
  • This kind of functionality was previously achieved using XMLHttpRequest which was a bummer to use.
  • Works in all modern browsers. caniuse.com

Intersection Observer API

  • Provides a way to observe changes in the intersection of a target element with another element.
  • Has a threshold option where you can indicate at what percentage of the target’s visibility the observer’s callback should be executed.
  • Works in all modern browsers. caniuse.com

URLSearchParams

  • Provides utility methods to work with the query string of a URL.
  • Manipulating and Retrieving Query Params is no longer a pain!
  • Works in all modern browsers. caniuse.com

Loading Attribute

  • Images aren’t loaded until a “user scrolls near” them.
  • Works with the following elements: img, picture, and iframe.
  • A simple html attribute that accepts three values: lazy, eager, and auto.
  • Currently only works in Chrome… caniuse.com

Document.designMode

  • Controls whether the entire document is editable.
    Why would you need this? /shrugs
  • Takes one of two values: “on” or “off” .
  • You can even use Cmd + B to bold text and Cmd + I to italicize text.
  • Works in all modern browsers… and IE11. caniuse.com
